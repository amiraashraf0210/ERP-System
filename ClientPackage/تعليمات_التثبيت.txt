         نظام ERP — دليل التثبيت
════════════════════════════════════════════

المتطلبات قبل التشغيل:
─────────────────────────────────────────
1. نظام تشغيل: Windows 10 أو أحدث
2. SQL Server Express (مجاني):
   https://www.microsoft.com/en-us/sql-server/sql-server-downloads
   اختر "Express" ثم ثبّته بالإعدادات الافتراضية

3. SQL Server Management Studio - SSMS (اختياري للإدارة):
   https://aka.ms/ssmsfullsetup

─────────────────────────────────────────
خطوات التثبيت:
─────────────────────────────────────────

الخطوة 1: ثبّت SQL Server Express
  - حمّل النسخة المجانية من الرابط أعلاه
  - اختر Basic Installation
  - احتفظ باسم السيرفر الذي يظهر بعد التثبيت
    (عادةً: COMPUTERNAME\SQLEXPRESS)

الخطوة 2: شغّل البرنامج
  - افتح ملف: ERP.UI.exe
  - ستظهر شاشة "إعداد قاعدة البيانات"

الخطوة 3: إعداد الاتصال
  - اسم السيرفر: COMPUTERNAME\SQLEXPRESS
    (استبدل COMPUTERNAME باسم جهازك)
  - اسم قاعدة البيانات: ERPSystem
    (أو الاسم الذي حدده لك المورد)
  - نوع المصادقة: Windows Authentication
  - اضغط "اختبار الاتصال" ثم "حفظ والمتابعة"

الخطوة 4: تفعيل البرنامج
  - ستظهر شاشة "تفعيل البرنامج"
  - أدخل كود التفعيل الذي حصلت عليه من المورد
  - اضغط "تفعيل"

الخطوة 5: تسجيل الدخول
  - اسم المستخدم: admin
  - كلمة المرور: 1234
  - غيّر كلمة المرور فور الدخول

─────────────────────────────────────────
للدعم الفني:
─────────────────────────────────────────
تواصل مع المورد للحصول على المساعدة.
